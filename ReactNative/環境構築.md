node,npmの確認後以下をインストール

## watchman をインストール
```sh
$brew install watchman
```

## expo-cli をインストール
```sh
$yarn global add expo-cli
```

## expoを動かす
任意のディレクトリでプロジェクトを作成
```sh
$expo init tommy-app
```
その後ディレクトリに移動し起動
```sh
$yarn start
```

## iOS simulator をインストール
- ブラウザに表示されたexpoの画面の「Run　on　iOS simulator」をクリック
- Xcodeを開きiOS simulatorをインストール（Xcodeを開かないといつまでたってもダウンロードが完了しないため)
